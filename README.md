# Travel Dashboard ✈️

Un dashboard interactiu per preparar el mateix viatge, amb dades en temps real sobre el temps atmosfèric i conversió de moneda.

## Funcionalitats
- **Selecció de ciutat**: Consulta dades de Barcelona, Londres, París, Nova York i Tòquio.
- **Dades de Temps en viu**: Utilitza l'API d'[Open-Meteo](https://open-meteo.com/) per obtenir la temperatura actual i la probabilitat de pluja.
- **Conversió de Moneda**: Integració amb l'API d'[Exchange Rate API](https://www.exchangerate-api.com/) per convertir EUR a la moneda local de la destinació.
- **Recomanacions Personalitzades**: Missatges dinàmics basats en les condicions meteorològiques.
- **Disseny Premium**: Creat amb un estil modern, glassmorphism i disseny responsive.

## Tecnologies utilitzades
- HTML5 Semàntic
- CSS3 (Variables, Flexbox, Grid, Glassmorphism)
- JavaScript Vanilla (Async/Await, Fetch API)
- Google Fonts (Inter & Outfit)

## Com utilitzar-lo
1. Obre el fitxer `index.html` en el teu navegador.
2. Selecciona una ciutat del desplegable superior.
3. Observa com les dades s'actualitzen automàticament.
4. Introdueix un import en EUR per veure la conversió a la moneda local.
